Frequently Asked Questions

What is the difference between HTML, CSS, and JavaScript?

HTML is the structure of the webpage, CSS is responsible for styling and layout, while JavaScript adds interactivity and dynamic behavior.

What is the benefit of using Semantic HTML?

Semantic HTML improves accessibility, SEO performance, and makes the code easier to read and maintain.

Why is Responsive Web Design important?

Responsive design ensures that websites look good and function properly across all devices, from mobile phones to large desktop screens.

What is the difference between class and id in HTML?

A class can be used on multiple elements, while an id must be unique on a page and is generally used for specific JavaScript targeting.

Why should you use External CSS instead of Inline CSS?

External CSS keeps code clean, improves maintainability, and allows you to apply the same styles across multiple pages.

What is Event Delegation in JavaScript and why is it useful?

Event Delegation is a technique where you attach a single event listener to a parent element instead of multiple children. It improves performance and works with dynamically added elements.

What is the purpose of the alt attribute in images?

The alt attribute provides alternative text for images. It improves accessibility for screen readers and helps with SEO.

Explain the CSS Box Model.

The CSS Box Model consists of content, padding, border, and margin. Understanding it is essential for controlling element spacing and layout.

What are the advantages of using Flexbox?

Flexbox makes it easier to design responsive and flexible layouts, especially for aligning items and distributing space within a container.

Why is it important to write clean and well-commented code?

Clean code improves readability, makes collaboration easier, helps with debugging, and shows professionalism to potential employers.
